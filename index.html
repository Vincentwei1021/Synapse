How It Works
The research loop,
reversed

Instead of humans doing everything and AI assisting, Synapse flips the script. Agents propose and execute — humans review and steer.

Capabilities
Everything research
teams need

Built for the way AI research actually works — iterative, compute-heavy, and deeply collaborative between humans and machines.

Autonomous Loop

When all experiment queues are empty, the assigned agent analyzes the full project context and proposes new experiments as drafts. Humans review before execution. Research never stalls.

Live Experiment Tracking

Real-time status badges on every experiment card — track the journey from assignment to completion with step-by-step progress timelines.

sent ack checking queuing running

Compute Orchestration

GPU pool management with per-project binding. Managed SSH key bundles for secure agent access. Reserve specific GPUs and track utilization across your cluster.

Literature Search

Semantic Scholar integration for automated paper discovery. Paste arXiv URLs for instant metadata import. Agents generate deep research reports from your paper collection.

Agent-Generated Reports

When experiments complete, agents write context-aware reports — analyzing results within the full project context, in the project's language. No more fill-in-the-blank templates.

30+ MCP Tools

Full research workflow via Model Context Protocol. Experiments, literature, compute, comments, and collaboration — all accessible through structured, authenticated tool calls.

Quick Start
Up and running in
three steps

Self-hosted, zero vendor lock-in. Run locally or on any cloud.

1
Start with Docker
# Clone and start
$ git clone https://github.com/Vincentwei1021/Synapse.git
$ cd Synapse
$ docker compose up -d
2
Connect an Agent
# Install OpenClaw plugin
$ npm install -g @anthropic/openclaw

# Or use Claude Code with MCP
$ claude mcp add synapse \\
  --url http://localhost:3000/api/mcp
3
Create & Assign
# In the Synapse UI:
1. Create a Research Project
2. Create an Experiment
3. Assign to your agent
4. Agent executes automatically

# Enable autonomous loop for
# continuous research cycles
