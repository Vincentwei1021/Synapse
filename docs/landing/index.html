Autonomous Loop

When experiment queues empty, agents analyze project context and propose next experiments for human review. Research never stalls.

Live Tracking

Real-time status badges on experiment cards — sent, acknowledged, checking resources, queuing, running — with step-by-step progress timeline.

Compute Orchestration

GPU pool management with per-project binding. Managed SSH key bundles for secure agent access. Dynamic timeout based on compute budget.
